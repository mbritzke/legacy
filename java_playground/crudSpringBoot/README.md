# Cars API
#### Features:

```
POST /car/
```
Add new car;

```
GET /car/{plate}
```
Return a car based on its license plate;

```
GET /car/
```
Return a list with all cars;

```
DELETE /car/{plate}
```
Deletes a car based on its license plate;

```
PUT /car/{plate}
```
Updates a car based on its license plate;
